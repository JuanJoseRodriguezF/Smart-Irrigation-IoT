# Spike Arquitectónico: Comunicación Bluetooth ESP32

## Objetivo
Evaluar la viabilidad de comunicación entre el ESP32 y una aplicación móvil utilizando Bluetooth Low Energy (BLE).

## Actividades realizadas
- Configuración de entorno ESP32
- Implementación de servidor BLE
- Envío de datos de prueba
- Conexión con app móvil

## Resultados
- Conexión exitosa con dispositivo móvil
- Envío de datos en tiempo real
- Estabilidad en la comunicación

## Problemas encontrados
- Desconexiones ocasionales
- Configuración inicial compleja

## Conclusión
El uso de BLE es viable para el proyecto. Se recomienda su implementación para el MVP.

## Decisión
Continuar con BLE como medio de comunicación principal.
